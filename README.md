### 关于项目介绍
这个项目是我打算用10年时间来操作维护的一个项目，从微小处入手，一步一步搭建一个大型综合系统，并且我会自己购买服务器和域名让他正常运行起来。其次我会把本项目中所有遇到的知识点都一一罗列出来并进行详解，一是为了让自己印象加深，二是让更多的程序员可以从中获取到点什么。
